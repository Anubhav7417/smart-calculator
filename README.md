Smart CalculatorThis is a versatile desktop calculator application built using Python's Tkinter library. It combines basic arithmetic operations with a currency converter, an age calculator, and a fun math fact display.‚ú® FeaturesBasic Calculator: Perform standard arithmetic operations (+, -, *, /, %).Currency Converter: Convert amounts between major currencies (USD, EUR, GBP, INR). Exchange rates are pre-defined within the application.Age Calculator: Calculate a person's age (years, months, days) based on their birth date. Includes validation for correct date input.Math Facts: Get a random interesting math fact with a click of a button.User-Friendly Interface: Intuitive graphical user interface (GUI) built with Tkinter.üöÄ Getting StartedPrerequisitesTo run this application, you need to have Python 3.x installed on your system.
